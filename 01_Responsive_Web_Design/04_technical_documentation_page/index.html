Introduction to this JavaScript Documentation
(This project was made as part of the Free Code Camp’s Responsive Web Design curriculum.)

A static website is one that is shown to the user exactly as stored, whereas a dynamic website is one that can change its layout or content, based on user input, time zone, settings etc. JavaScript is the language that makes websites dynamic.

For this documentation, it is convenient to have:

  • some basic knowledge on how the web works,
  • and some working knowledge of HTML and CSS.
Hello World

This code displays 'Hello, World' as an alert in the browser:

alert('Hello, World')

And this code displays 'Hello, World' in the browser console (press F12 in your browser to open the console, or right-click anywhere on the page > click the 'inspect' option > click the 'console' tab.)

console.log('Hello, World')
